# AGENTS.md

## Goal
이 저장소를 분석하여 초보자도 이해할 수 있는 README.md를 작성한다.

## Documentation Rules
- README는 한국어로 작성한다.
- README에는 반드시 다음 섹션을 포함한다:
  1. 프로젝트 개요
  2. 폴더 구조
  3. 설치 방법
  4. 실행 방법
  5. 테스트 방법
  6. 주요 함수 설명
- 명령어는 모두 코드 블록으로 작성한다.
- 설명은 짧고 명확하게 작성한다.

## Project Understanding
- src/calculator.py의 각 함수 역할을 설명한다.
- src/main.py를 바탕으로 프로그램 실행 방법을 설명한다.
- tests/test_calculator.py를 바탕으로 테스트 실행 방법을 포함한다.

## Completion Criteria
- README.md 파일이 생성되어 있어야 한다.
- README에 프로젝트 개요, 폴더 구조, 설치 방법, 실행 방법, 테스트 방법이 모두 있어야 한다.
- 폴더 구조는 트리 형태로 작성한다.
